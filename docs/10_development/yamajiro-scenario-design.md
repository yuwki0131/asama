# 山城シナリオ設計: 霞ヶ峰城

## 概要

霞ヶ峰城 (scenario id: `mountain-castle`) は Release 2.0 のショーケースシナリオ。
自然の岩山に三段の石垣曲輪を積み上げた架空の山城で、高低差システム (elevation-contract.md) の
実戦デモとして設計された。坂だけが山上への経路であり、高所の弓兵が常に斜面を支配する。

参考: 岩村城 (多段石垣)、高取城 (長い大手道)、備中松山城 (岩肌+石垣)。

## 高低差レイアウト

| 段 | 曲輪     | 種別 | 範囲 (approx)              | スキン   |
|----|----------|------|---------------------------|----------|
| L3 | 本丸     | rect | x=51..62, y=57..64         | ishigaki |
| L2 | 二の丸   | rect | x=47..68, y=55..71         | ishigaki |
| L1 | 三の丸   | rect | x=44..73, y=52..85         | ishigaki |
| L1 | 山体岩肌 | ellipse | cx=58, cy=66, rx=21, ry=17 | cliff  |
| L0 | 城下     | flat | y >= 87                    | grass    |

高い段の patch が低い段を上書き (max-composition)。L2 patch の上に L3 patch が乗る。

## 登城ルート (坂)

坂以外の曲輪縁はすべて石垣崖 (impassable)。

| 坂         | position    | toward | width | 接続     |
|------------|-------------|--------|-------|---------|
| 大手道 下段 | (56, 86)    | N      | 2     | L0 → L1 |
| 搦手道 下段 | (74, 79)    | W      | 1     | L0 → L1 |
| 大手道 中段 | (52, 72)    | N      | 2     | L1 → L2 |
| 搦手道 中段 | (68, 72)    | N      | 1     | L1 → L2 |
| 虎口       | (60, 65)    | N      | 1     | L2 → L3 |

## 勝利条件

本丸を 24000 ticks (20 分) 保持し続ければ守備側の勝利。

## 波状攻撃 AI

| 波  | tick   | 概要                                          |
|-----|--------|-----------------------------------------------|
| 第1 | 3000   | 大手前哨: 槍のみ。大手坂・惣構の防御を観客に見せる。 |
| 第2 | 7200   | 大手主力: 槍+弓。城下柵突破を想定。             |
| 第3 | 12000  | 二正面開始: 大手継続 + 搦手(南東)に騎馬側面隊。  |
| 第4 | 17000  | 総攻撃: 大手・搦手同時、鉄砲援護+工兵+荷車2台。  |

Wave 3 から二正面化し、第4波で虎口前まで到達可能な最大圧力。
