# リリース1.0 スコープ(2026-07-05 ユーザー裁定・確定)

## ビジョン

- **1.0 = 防衛シナリオ特化**。「城と城下町を作り、守り切る」体験の完成度に全振りする。
- 攻城側プレイは2.0候補。**マルチプレイヤー・MOD対応は恒久スコープ外**。キャンペーン(連戦)は2.0候補。

## 1.0 に含めるゲームシステム

### 攻撃側兵糧(最優先)
- 敵軍は**補給荷車**を伴って侵攻する。
- プレイヤーが荷車を破壊すると**撤退タイマー**が発動し、制限時間内に敵が本丸を落とせなければ
  **敵は撤退確定=プレイヤー勝利**(勝利理由: 兵站切断)。
- ※衰弱(攻撃力・速度低下)は実装しない(ユーザー裁定で簡素化)。
- 数値(タイマー長・荷車HP等)はバランス値として仮置き→プレイテストで調整。

### 兵種相性(3すくみ+α)
- 槍足軽 = 対騎兵 1.5x
- 騎兵 = 対遠隔(弓・鉄砲)1.5x、移動速度 約2x
- 弓・鉄砲 = 対歩兵有利
- 倍率はすべてデータ値。

### 新兵種3種
- **鉄砲兵**: 射程4・高攻撃力・長クールダウン・近接に弱い
- **騎兵**: 高速・突撃ボーナス・堀と塀を越えられない(梯子も使用不可)
- **補給荷車**: 非戦闘・攻撃側兵糧の供給源(敵専用)

### 土塁・石垣
- **建造物方式**(塀と同じ接続キットの新BuildingType)。地形改変方式は不採用。
- 土塁 = 安価・低HP・(2.0で射線導入時に「弓は越えて撃てる」を検討)
- 石垣 = 高価・高HP

## 1.0 に含めるコンテンツ・UI

- **シナリオ3本**: A 環郭式平城(入門)/ B 連郭式(標準)/ C 川城+門前町(上級)。
  `tools/scene-preview/` の3レイアウトをシナリオ化し、AIプロファイル
  (正面強襲/梯子強襲/混合)を対応付ける。
- **UI**: 選択情報パネル/グループ登録(Ctrl+数字)/ダブルクリック同兵種選択。
- **アート**: 天守は「現AIアート vs 3D+ペイントオーバー版」の比較判定を実施(統括が比較画像を制作)。

## バランス数値の方針

- 蔵容量・徴税率・人口増減・梯子HP等の細かい数値は**決めずにデータ値のまま**とし、
  3シナリオ完成後の**プレイテストフェーズで実測調整**する(ユーザー裁定)。
