# ドメインモデル

> 技術的なTypeScript型は技術設計章で確定します。本書は概念責務を定義します。

## World

- 現在tick
- 暦
- ゲーム速度
- マップ
- 勢力
- エンティティ
- シナリオ状態

## Map

- 幅・高さ
- TerrainCell配列
- 本丸セル
- 出現地点
- 外部搬入口
- 建築制限

## Unit

- ID
- 所属
- 兵種
- HP
- 位置
- 現在命令
- 経路
- 行動状態

## Building

- ID
- 所属
- 種類
- HP
- 位置・向き
- 占有セル
- 破壊状態

## Honmaru

- セル集合
- 内部の勢力別占領可能ユニット数
- 接続蔵
- 利用可能兵糧

## Storehouse

- 食料
- 容量
- 本丸接続状態
- 本丸距離順位

## SupplyCart

- 所属
- 食料
- 最大積載量
- 防衛側格納先命令

## PopulationState

- 人口
- 人口上限
- 支持率
- 徴税率
- 農業基盤
- 徴兵可能人数
- 最大徴兵人数

## ResourceState

- 銭
- 木材
- 石材
- 武器

## ScenarioState

- 現在フェーズ
- 攻城状態
- 発生済みイベント
- 次イベント
- 勝敗状態
