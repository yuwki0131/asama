# AIプロファイル

## 基本方針

MVPでは万能な戦術AIを作らず、シナリオから行動プロファイルを選択します。

## 正面強襲型

1. 最小移動コストで到達可能な門を評価
2. 近接兵を門へ移動
3. 門を攻撃
4. 破壊後、本丸へ進軍

## 梯子強襲型

1. 梯子設置可能な城壁を列挙
2. 本丸までの想定コストを評価
3. 工兵を設置地点へ移動
4. 梯子設置
5. 歩兵を投入
6. 本丸へ進軍

## 堀埋め型

1. 目標門または梯子地点を選択
2. 必要な堀セルを抽出
3. 工兵へ堀埋め命令
4. 完了後に強襲

## 混合型

- 門攻撃と梯子を複数地点で実行
- 一部工兵で堀埋め
- 兵力配分はシナリオ設定

## 防衛AI

攻撃側プレイ用シナリオで使用します。

- 門前へ近接兵
- 櫓・城壁へ遠距離兵
- 侵入地点へ増援
- 本丸内に最低限の守備兵を維持

## 非対象

MVPでは以下を行いません。

- 学習型AI
- 高度な欺瞞
- 長期的な経済最適化
- Fog of War前提の索敵
- 火災・砲撃判断
