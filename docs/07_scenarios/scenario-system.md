# シナリオシステム

## シナリオで設定する項目

- マップID
- プレイヤー勢力
- 初期資源
- 初期人口・支持率・徴兵枠
- 初期建物
- 初期ユニット
- 本丸モード
- 攻城開始・終了時刻
- 敵出現地点
- 補給荷車
- 外部支援
- AIプロファイル
- 勝敗条件
- シナリオ終了時刻

## 本丸

- `player_select`: 開始時にプレイヤー指定
- `predefined`: シナリオに固定

## 攻城イベント

攻城状態はシナリオイベントで明示的に変更します。

例:

```text
時刻19分: 第1次攻撃開始
敵全滅または兵糧切れ: 攻城終了
時刻52分: 第2次攻撃開始
```

## 補給荷車

シナリオごとに以下を設定します。

- 所属
- 出現地点
- 積載量
- 出現時刻
- 条件
- 回数

## 勝敗条件

組み合わせ可能:

- 本丸占領
- 防衛側兵糧ゼロ
- 攻撃側兵糧ゼロ
- ユニット全滅
- 規定時刻まで防衛

## スクリプト方針

MVPでは汎用スクリプト言語を導入せず、定義済みイベント種別と条件をJSON等で記述します。
