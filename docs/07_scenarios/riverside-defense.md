# 川沿いの城（Riverside Defense）

難易度: **C — 上級**（release-1.0-scope C1 シナリオ3本体系）

## コンセプト

川を天然の堀とする非対称防衛戦。攻撃側は橋2ヶ所を渡らなければ城に近づけず、橋は守備の要衝になる。5波構成の最高難度シナリオ。

波4以降に**鉄砲兵・騎兵・補給荷車**が登場し、兵種相性と荷車迎撃の両方を要求する。

## 地形構造

```
西岸（守備側）          川(x=58)    東岸（攻撃側）
x=24~52               x=58        x=60~95

農地  [天守]            |水堀|
      [本丸]  [城壁]    |木橋|y=50  ← 北橋
      [兵舎]  [城壁]    |水堀|
              [三門]    |水堀|   (主門: y=56-58)
      [蔵]    [城壁]    |水堀|
      [市場]  [城壁]    |土橋|y=66  ← 南橋
      [居館]  [城壁]    |水堀|
      [町区]            |水堀|
```

- 川(x=58): 水堀13タイルでほぼ全面を遮断
- 北橋(y=50): 木橋。耐久力低く破壊されやすい危険な渡河点
- 南橋(y=66): 土橋。耐久力高い。南回りルートの要所

## 守備設計

### 二段構えの守備線

1. **橋頭堡の門(x=56)**: 北・南の橋出口それぞれに門を設置。橋を渡った直後に足止め
2. **城壁(x=52)**: 東向きに連続する高耐久壁。y=44-70を連続防御し、中央(y=56-58)に3マス縦門を開ける

### 非対称性の活用

- 北橋は城の北側(y=50)につながるため、本丸(y=57)まで南下する必要がある
- 南橋は城の南側(y=66)につながり、城壁南部を直接圧迫する
- 北から攻めれば城壁側面を取られ、南から攻めれば本丸ルートが遠回り
- 守備側は弓兵2名を橋頭堡に配置するか、城壁越しに集中するか判断を迫られる

## 攻め筋（AIの行動パターン）

**前半（波1-2）**: 正面強襲型。北橋に集中し、橋頭堡の門を攻撃する。荷車は後方に待機。

**中盤（波3）**: 工兵が水堀を埋めようとし始める（`fillMoat`タスク）。並行して南橋から別動隊を送り、守備側の注意を分散。荷車を守るため槍兵が護衛。

**後半（波4-5）**: 南北同時に橋を渡る圧力戦術。工兵は城壁に梯子を掛けようとする（`ladder`タスク）。鉄砲兵が長射程で城壁守備兵を制圧し、騎兵（波5）が高速で城内に突入を図る。荷車2台の並行迎撃が最終課題。

## 設計上の選択

- **木橋と土橋の使い分け**: 北橋(木橋)は耐久力が低く工兵の破壊対象になりやすい。プレイヤーが守らないと橋が落ちて北から攻めにくくなるが、工兵なしでは南橋が主ルートになる
- **橋頭堡の門**: 城壁手前の第一防衛線。これが破られると城壁への圧力が高まる。プレイヤーが積極的に守る位置として機能
- **3マス縦門(gate_wide_3_ne_sw)**: 城壁中央の主門。敵が城内に侵入した際の最後の関門
