# 環郭の城（Concentric Castle）

難易度: **A — 入門**（release-1.0-scope C1 シナリオ3本体系）

## コンセプト

同心円状に配置した防衛リングで南から進軍する敵を食い止める入門シナリオ。
プレイヤーは **建設→徴兵→防衛→荷車撃破** の基本サイクルを体験する。

波は3波・ゆったりした間隔。最大の教育目標は **補給荷車(supply_cart)を破壊して撤退タイマー(兵站切断 victory)を発動させる体験**。

## ファーストプレイ体験の設計意図

このシナリオはプレイヤーが以下を順番に学ぶよう設計されている:

1. **建設フェーズ** — 第1波まで約3分間。柵・壁・蔵の追加建設で防衛線を固める。
2. **徴兵フェーズ** — 兵舎から槍足軽や弓兵を徴用し守備を増強する。
3. **防衛フェーズ** — 南から進軍する敵を二之丸南門→土橋→内郭南門の多重チョークで迎撃する。
4. **荷車撃破フェーズ** — 各波に同行する補給荷車を弓で狙い、撤退タイマーを発動させる。荷車はHPが低く移動が遅いため比較的狙いやすく、撃破の達成感をビギナーに与える。
