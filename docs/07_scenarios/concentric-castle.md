# 環郭の城（Concentric Castle）

難易度: **A — 入門**（release-1.0-scope C1 シナリオ3本体系）

## コンセプト

同心円状に配置した防衛リングで徐々に守備陣地を後退させながら戦う入門シナリオ。
東から接近する敵に対して **「東門柵→水堀リング→内郭石垣」** の3段階防衛線が機能する。

波は3波・緩め。最大の教育目標は**補給荷車(supply_cart)を破壊して撤退タイマーを発動させる体験**。

## 地形構造

```
西（背面山野）        本丸(x=30-43)      水堀リング       東柵門(x=54)   東（敵来方向）
                    ┌─────────────┐
                    │ [天守]      │
              ~~~~~ │  [本丸]    │ ~~~~~  ─── 柵・門 ───  → 敵×100
                    │            │
                    └──────┬──────┘
                           土橋(y=45)
       [城下町・農地 — 南 y=48+]
```

| 防衛線 | 座標 | 種別 | 役割 |
|--------|------|------|------|
| 東柵門 | x=54, y=36 | gate_ne_sw + fence | 第一チョークポイント |
| 水堀東辺 | x=45, y=27-44 | water_moat | 敵を南回りに誘導 |
| 土橋 | x=35, y=45 | earth_bridge | 水堀唯一の渡河点 |
| 内郭南門 | x=34-36, y=43 | gate_wide_3 | 最終チョークポイント |

## 守備設計

### 3段防衛の仕組み

1. **東柵門 (x=54)**: 敵の侵入路を1点に絞るフェンス+門ライン。弓兵を柵の北南に配置して遠距離で削る。
2. **水堀リング**: water_moat は不可通過(passable=false)。敵は東辺を迂回し南の土橋へ誘導される。
3. **内郭南門**: 土橋を渡った後の最終関門。槍足軽で固めて剣足軽の突破を防ぐ。

### 荷車破壊の体験

各波に supply_cart が1台同行する。荷車は遅く攻撃力ゼロなので比較的狙いやすい。
破壊すると **撤退タイマー(supply_cut victory)** が発動し、残敵が本丸を落とせなければプレイヤー勝利。
入門レベルでこのメカニクスを体得させるのが本シナリオの核心。

## 攻め筋（AIの行動パターン）

**AIプロファイル: 正面強襲型**（東から一方向）

東 x=100 から直進し、東柵門(x=54)を攻撃する。
水堀で阻まれた後は南の土橋(x=35, y=45)を経由して本丸を目指す。
工兵なし・騎兵なし → プレイヤーが「堀」「橋」「門」の概念に慣れやすい。
