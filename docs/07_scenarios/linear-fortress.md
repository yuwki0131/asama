# 連郭の城（Linear Fortress）

難易度: **B — 標準**（release-1.0-scope C1 シナリオ3本体系）

## コンセプト

北西の本丸と東の二之丸を空堀+土橋で連結した連郭式城郭。**東からの正面攻撃と南からの騎兵迂回**という二方向圧力に守備側が対処する標準難度シナリオ。

4波構成。波3以降の騎兵フランキングが守備配置の決定を迫る。

## 守備設計

### 連郭の意義

1. **二之丸を第一防衛線** として展開: 兵舎・倉が二之丸内にあり増援しやすい。
2. **空堀が縦断防御壁**: 空堀は移動コスト×5(減速)。工兵なしでは実質的な障壁。
3. **本丸は最終砦**: 空堀と土橋で時間を稼ぎ、二之丸が破られても籠城できる。

### 騎兵フランキングへの対処

波3以降、騎兵が南路(y=100 付近)から迂回してくる。
騎兵は移動速度2倍(ticksPerStep=3)だが、水堀・土塁は越えられない。
南の城下町側には门がないため、弓兵を城下町道路に配置するか騎兵を迎撃する判断が必要。

## 攻め筋（AIの行動パターン）

**AIプロファイル: 混合型**（東正面強襲 + 騎兵南迂回）

**前半（波1-2）**: 東道路から二之丸東門を攻撃。弓兵が門を遠距離射撃で支援。
波2から南プローブが城下町南端から侵入を試みる。

**中盤（波3）**: 工兵が空堀の土橋周辺を狙う(`fillMoat`タスク)。
騎兵が南から高速で城下町を突っ切り、本丸への近道を探す。

**後半（波4）**: 東と南が同時展開。鉄砲兵が二之丸東門外から射撃支援。
荷車が2台になり、守備側は北と南で並行して兵站を断つ必要がある。
