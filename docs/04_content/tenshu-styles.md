# 天守様式

## 位置付け

天守は城の象徴的建築であり、ゲーム上は大型の防御・射撃施設です。

- 建築は任意
- 本丸内に配置
- 勝敗条件ではない
- 天守が存在しない城も正規の構成

## 外観バリエーション

将来的に以下の「実在城風」様式を用意します。

- 犬山城風: 小型・古風
- 松本城風: 黒基調・重厚
- 姫路城風: 白漆喰・大型・華麗
- 松江城風: 黒基調・質実剛健
- 松山城風: 山上・立体的

## MVP

- 最初は1種類でも成立
- アセットパイプライン検証後に3種類へ拡張
- 最終的に5様式を目標

## 性能

MVPでは外観差を中心とし、性能差は小さくします。

共通候補:

- 高HP
- 兵士収容
- 高所補正
- 高コスト

威信・支持率・視界効果は後から追加可能です。
