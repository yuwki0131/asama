## 初期相性

- 槍足軽 → 騎兵: 1.5倍
- 刀足軽 → 弓兵・鉄砲兵: 1.2倍
- 騎兵 → 弓兵・鉄砲兵: 1.4倍
- 弓兵 → 騎兵: 0.8倍
- 鉄砲兵 → 侍・騎兵: 1.25倍
- 工兵: 全戦闘で低性能
