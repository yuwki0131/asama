# ゲームコンセプト

## 概要

日本の応仁の乱期から江戸時代初期をイメージした、築城・城下町造営・攻城・籠城・野戦を扱うリアルタイムストラテジーゲームです。

Stronghold風の分かりやすい個兵操作と即時建築を採用しつつ、日本城郭固有の以下の要素を中心に据えます。

- 本丸
- 門・虎口
- 石垣・土塁
- 空堀・水堀
- 櫓・天守
- 城内外の蔵
- 兵糧攻め
- 梯子と堀埋め

## 基本ループ

1. 本丸を設定する
2. 城郭を築く
3. 町区画・農地・道路を整備する
4. 徴税と人口増加によって銭・徴兵余力を得る
5. 兵士を登用する
6. 秋の収穫や市場で兵糧を確保する
7. 敵の攻城を受ける、または敵城を攻める
8. 被害と弱点を確認して城を改修する

このサイクルを1プレイ中におおむね3回繰り返します。

## 1プレイの想定

- 1倍速換算で約90分
- ゲーム内では約6年
- 築城、内政、攻城の比率をおおむね均等に配分
- 時間速度は一時停止、1倍、2倍、4倍

## 歴史表現

本作は史実再現ではなく歴史風ゲームです。

- 時代は応仁の乱前後から江戸時代初期まで
- 中世山城、戦国城郭、近世城郭の要素を同時に利用可能
- 実在城の厳密な再現を要求しない
- 天守がない城も正規の城として成立する
- 天守は勝敗条件ではなく、大型櫓に近い任意施設

## 勝敗の中核

防衛側は次のいずれかで敗北します。

- 本丸陥落
- 本丸から利用可能な兵糧がゼロ

攻撃側は次のいずれかで敗北します。

- 戦闘可能ユニット全滅
- 補給荷車に保持される兵糧がゼロ

このため本作は、単純な殲滅戦ではなく、築城と兵糧管理を軸とした攻城RTSです。
