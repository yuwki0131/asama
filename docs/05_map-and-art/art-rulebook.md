# アートルールブック(ビジュアルQAゲート判定基準)

ユーザーフィードバックから抽出した**チェック可能なルール**の集約。各ルールはID付きで、
L1(機械lint: `assets:lint:art`)または L2(VLMセルフレビュー)のどちらで検証するかを明記する。

運用原則: **一度指摘されたことは二度とユーザーに指摘させない。**
指摘→修正のサイクルは、該当ルールをここに1行追加してからクローズする(→ [PROC-01])。

様式語彙(ペインタリー、植生様式シート、品質標準)は `art-direction.md` が正。
本書はそれを「合否判定できる文」に変換したチェックリストである。

---

## GEO: 幾何契約(L1機械検査)

| ID | ルール | 検証 |
|---|---|---|
| GEO-01 | 建物キャンバスはH-32ルール: `anchor.x=0.5`、`anchor.y*height = height-32`(アンカーはフットプリント菱形中心、下端から32px上) | L1: manifest全件照合 |
| GEO-02 | 地形面アセット(cliff/ishigaki face・corner)はキャンバス `64×(32+40h)`、アンカー `(0.5, 16/(32+40h))` | L1: manifest照合 |
| GEO-03 | 面アセットの**壁上端はセル辺に一致**すること: 上端行(y16〜24)の不透過x範囲が e面=右半分(x≥32)/ s面=左半分(x≤32)に収まり、辺からのドリフト≤2px。裾の反り出しは合計≤13px(石垣sori上限0.40 world units) | L1: 行ごと不透過x範囲走査 |
| GEO-04 | オートタイルファミリ(橋・川岸・道等)はタイル境界1px列の色シグネチャが隣接バリアント間で連続すること | L1: 境界列比較 |
| GEO-05 | 基壇・砂利パッドはフットプリント菱形と幾何学的に一致(はみ出し・不足なし) | L1: bbox/菱形適合 + コンタクトシート |
| GEO-06 | アニメシートは列=フレーム、行=方向。フレーム間でアンカーの重心ドリフトなし | L1 |

## TONE: トーン・経年感

| ID | ルール | 検証 |
|---|---|---|
| TONE-01 | 種別ごとのトーン窓(天守実測基準の meanLuma/meanSat)から逸脱しない。「白すぎ・新品」は不合格 | L1: aiIntakeのToneStats流用 |
| TONE-02 | 経年感必須: 雨だれ縦筋・壁裾の跳ね汚れ・壁足元の湿り/苔グラデーション等、風化記号を最低1系統 | L2 |
| TONE-03 | ペインタリーランプ(暖光×青紫影、光源=画面左上固定)。清潔なCG的グラデ禁止 | L2 |

## 種別ルール(L2レビューカード)

### 田畑(FARM)

| ID | ルール |
|---|---|
| FARM-01 | 稲は**1px幅の線ストローク**を株元から扇状に立て先端を尖らせる。塗り潰し楕円・円形クランプ・「もこもこの塊」は禁止 |
| FARM-02 | 夏=緑の葉+明るいチップ、秋=穂が横下に垂れるが**線的質感を維持**(黄色い塊化禁止) |
| FARM-03 | 実際の稲の風景(夏秋の水田)を参照前提として判定する |

### 天守(TENSHU)

| ID | ルール |
|---|---|
| TENSHU-01 | シルエット h/w 比 1.0〜1.2(櫓と同等の塔状比率)。「太い」印象は不合格 |
| TENSHU-02 | 最上層屋根幅 ≤ 1層目屋根幅の55% |
| TENSHU-03 | 石垣マウンドは切込接ぎの滑らかな4面。四隅の算木積み突起(quoinブロック)禁止 |

### 石垣(ISHIGAKI)

| ID | ルール |
|---|---|
| ISHIGAKI-01 | 反り(sori)振幅は絶対上限0.40 world units(高さ比例で伸ばさない)。壁上端はセル辺にピン留め(→GEO-03) |
| ISHIGAKI-02 | 切込接ぎの目地: 水平目地は通し、縦目地は段ごとに破る。積み重ね/隣接タイル間で目地(8px/段)が連続 |

### 門(GATE)

| ID | ルール |
|---|---|
| GATE-01 | 開閉の視覚区別を明確に: 開=中央通路の地面が見通せる、閉=暗い扉パネルで完全に塞がる |

### 植生(VEG)

| ID | ルール |
|---|---|
| VEG-01 | 判定は `art-direction.md` 植生様式シートへの適合。「本物っぽいか」での判定は**禁句** |
| VEG-02 | 写実への接近(針葉・葉ノイズの再現)は禁じ手 |

## PROC: プロセス規律

| ID | ルール |
|---|---|
| PROC-01 | ユーザー不合格が出たら、機械化可能なら L1 ルール、審美なら本書のカードに**必ず1行追加**してからサイクルをクローズ |
