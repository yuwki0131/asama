# アートルールブック(ビジュアルQAゲート判定基準)

ユーザーフィードバックから抽出した**チェック可能なルール**の集約。各ルールはID付きで、
L1(機械lint: `assets:lint:art`)または L2(VLMセルフレビュー)のどちらで検証するかを明記する。

運用原則: **一度指摘されたことは二度とユーザーに指摘させない。**
指摘→修正のサイクルは、該当ルールをここに1行追加してからクローズする(→ [PROC-01])。

様式語彙(ペインタリー、植生様式シート、品質標準)は `art-direction.md` が正。
本書はそれを「合否判定できる文」に変換したチェックリストである。

## GEO: 幾何契約(L1機械検査)

| ID | ルール | 検証 |
|---|---|---|
| GEO-01 | 建物アンカーは族別幾何契約に従う(レンダラーのピン位置準拠): 標準建物ロット=H-32(`anchorRow=height-32`、非正方フットプリントは `anchorX=width/2+(fw-fh)*16` の半タイルオフセット)/ 接地タイルキット(道・柵・塀・本丸マーカー・deco)=`height-16` / 堀=上端から16px(掘り込みは下方向に延長)/ 門=`height-8-(n+1)*8`(n=セル数)/ 橋=材質定数(土24px・木32px、下端から)/ 平地ロット(田)=キャンバス中心かつ `width=2*height` | L1: manifest全件照合 |
| GEO-02 | 地形面アセット(cliff/ishigaki face・corner)はキャンバス `64×(32+40h)`、アンカー `(0.5, 16/(32+40h))` | L1: manifest照合 |
| GEO-03 | 面アセットの**壁上端はセル辺に一致**すること: 上端行(y16〜24)の不透過x範囲が e面=右半分(x≥32)/ s面=左半分(x≤32)に収まり、辺からのドリフト≤2px。裾の反り出しは合計≤13px(石垣sori上限0.40 world units) | L1: 行ごと不透過x範囲走査 |
| GEO-04 | オートタイルファミリ(橋・川岸・道等)はタイル境界1px列の色シグネチャが隣接バリアント間で連続すること | L1: 境界列比較 |
| GEO-05 | 基壇・砂利パッドはフットプリント菱形と幾何学的に一致(はみ出し・不足なし) | L1: bbox/菱形適合 + コンタクトシート |
| GEO-06 | アニメシートは列=フレーム、行=方向。フレーム間でアンカーの重心ドリフトなし | L1 |
| GEO-07 | 描画順: 隣接建物のスプライトが他建物の屋根庇を上書き(見切れ)しないこと。等深度(maxX+maxY同値)のフットプリントは物理的に遮蔽し得ないため、庇の張り出しが大きいもの(面積大)を後描きする | renderGeometry.test.ts回帰 + L2目視(拡大時の庇連続性) |

## NOISE: ノイズ・エッジ品質(L1機械検査)

| ID | ルール | 検証 |
|---|---|---|
| NOISE-01 | 孤立α連結成分(スペックル、<4px)禁止。UIオーバーレイ(kind=overlay)は対象外(破線・コーナーティックが意匠) | L1 |
| NOISE-02 | 輪郭帯の黒マットフリンジ禁止(輪郭1〜2px帯の暗色率閾値) | L1 |
| NOISE-03 | 不透過本体内部の透明穴禁止。UIオーバーレイ(kind=overlay)は対象外(中空リング・枠が意匠) | L1 |
| NOISE-04 | 輪郭エッジ帯(屋根軒先等)の高コントラスト市松/ダッシュ列(明暗交互1px、クリーンアップ残滓)禁止。不透過本体内部の半透明明色ピクセルも不可 | L2 |

## TONE: トーン・経年感

| ID | ルール | 検証 |
|---|---|---|
| TONE-01 | 建物はロット建物実測トーン窓(art-direction.md「建物トーン窓」: luma3バンドのmean rgb/warm/sat)から逸脱しない。無彩色〜青寄り(warm<+5)の漆喰・瓦は不合格(石材帯はISHIGAKI-03で例外)。「白すぎ・新品」も不合格 | L1: aiIntakeのToneStats流用 |
| TONE-02 | 経年感必須: 雨だれ縦筋・壁裾の跳ね汚れ・壁足元の湿り/苔グラデーション等、風化記号を最低1系統 | L2 |
| TONE-03 | ペインタリーランプ(暖光×青紫影、光源=画面左上固定)。清潔なCG的グラデ禁止 | L2 |

## 種別ルール(L2レビューカード)

### 田畑(FARM)

| ID | ルール |
|---|---|
| FARM-01 | 稲は**1px幅の線ストローク**を株元から扇状に立て先端を尖らせる。塗り潰し楕円・円形クランプ・「もこもこの塊」は禁止 |
| FARM-02 | 夏=緑の葉+明るいチップ、秋=穂が横下に垂れるが**線的質感を維持**(黄色い塊化禁止) |
| FARM-03 | 実際の稲の風景(夏秋の水田)を参照前提として判定する |
| FARM-04 | 実った水田の床は「日陰の藁ゴールド」であり土ではない。床と株のコントラストを低く保つ(高コントラストは塊/ドット読みを誘発しFARM-01違反に繋がる) |
| FARM-05 | 稲穂は閉じたキャノピーの上に**最上層として後描き**する: 連続した1pxアーチ(立ち上がり→垂れ)、田全体で風向バイアス(≈85%同方向)、穂ストロークにピクセルノイズを乗せない(ノイズはアーチを紙吹雪化させ判読性を殺す) |
| FARM-06 | 季節派生は元絵の近黒ピクセル(水際テクスチャ)を田内部に残さない(ゴールド地に黒コショウ斑となる)。畦のアウトライン黒は保持 |

### 天守(TENSHU)

| ID | ルール |
|---|---|
| TENSHU-01 | シルエット h/w 比 1.0〜1.2(櫓と同等の塔状比率)。「太い」印象は不合格 |
| TENSHU-02 | 最上層屋根幅 ≤ 1層目屋根幅の55% |
| TENSHU-03 | 石垣マウンドは切込接ぎの滑らかな4面。四隅の算木積み突起(quoinブロック)禁止 |

### 石垣(ISHIGAKI)

| ID | ルール |
|---|---|
| ISHIGAKI-01 | 反り(sori)振幅は絶対上限0.40 world units(高さ比例で伸ばさない)。壁上端はセル辺にピン留め(→GEO-03) |
| ISHIGAKI-02 | 切込接ぎの目地: 水平目地は通し、縦目地は段ごとに破る。積み重ね/隣接タイル間で目地(8px/段)が連続 |
| ISHIGAKI-03 | 石積み様式は全アセットで統一: 建物マウンド(天守・櫓等)も地形石垣と同一の切込接ぎ(段高・目地・トーン・経年感)。別様式(野面積み風・塗りマウンド等)の混在はパッチワーク感を生み不合格 |

### 門(GATE)

| ID | ルール |
|---|---|
| GATE-01 | 開閉の視覚区別を明確に: 開=中央通路の地面が見通せる、閉=暗い扉パネルで完全に塞がる |

### 橋(BRIDGE)

| ID | ルール |
|---|---|
| BRIDGE-01 | 木橋は太鼓橋の反り(sori)を持つ: 両端で立ち上がり、中央部は頂部で水平。土橋は平坦な土手道のまま(反り禁止) |
| BRIDGE-02 | 橋の長さは河川/堀の実幅に自動適合する(下限3セル・上限なし)。start/mid/end セグメントのタイリングは継ぎ目・段差なく接続 |

### 植生(VEG)

| ID | ルール |
|---|---|
| VEG-01 | 判定は `art-direction.md` 植生様式シートへの適合。「本物っぽいか」での判定は**禁句** |
| VEG-02 | 写実への接近(針葉・葉ノイズの再現)は禁じ手 |
