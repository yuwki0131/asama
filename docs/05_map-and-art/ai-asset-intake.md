# AIアート・アセット取り込み仕様(ハイブリッド方式)

2026-07-05確定。建築物・地形・接続キット=手続きBlender、**有機的な単独装飾(木・藪・岩など)=AIアート**のハイブリッド方式。
AIアートは「ピクセル契約(タイル整合・接続マスク・アンカー精度)が不要な対象」に限って使用する。

## ワークフロー

```
AI生成(人間) → assets/source/ai/ に配置
  → pnpm run assets:ai-intake        # 切り抜き・正規化・トーン較正
  → assets/source/raster/ に出力     # 以降は既存のrasterパイプラインが処理
  → production-assets.json に登録(source.type = "raster")
  → pnpm run assets:all              # 通常ビルド
```

**白い被写体の注意(2026-07-05)**: 漆喰建築など白基調のアセットは、
標準ゲートの「閉域白ポケット除去・白フリンジ浸食」が被写体を侵食する
(天守で白壁に1,472pxの透明穴が開いた実例)。白い被写体は
**境界フラッドフィルのみ+明度235以上の境界限定デフリンジ2回**で取り込むこと。

取り込みゲート(assets:ai-intake が自動実行):
1. **背景除去** — 四隅からのフラッドフィルで単色背景を透過化(既に透過ならスキップ)
2. **トリム＆正規化** — 不透明領域をトリムし、指定キャンバスへ「幅fit・底辺=アンカー」で配置
3. **トーン較正** — 天守AIアート(品質基準)の実測統計に平均輝度・彩度を線形一致させ、世界のトーンから浮かないようにする
4. **検証** — 出力の寸法・アンカー接地・アルファ占有率をチェックし、比較プレビュー(`assets/intermediate/ai-intake/`)を出力

## 生成プロンプト仕様(共通)

- スタイル: 「日本の戦国〜江戸初期の城下町を描いたペインタリーなゲームアセット。落ち着いた暗めの暖色トーン、経年感のある自然な描写」
- 参照: 既存の天守AIアートとトーン・筆致を揃える(可能ならスタイル参照として添付)
- **ライティング: 画面左上からの柔らかい光**(既存アセットと統一)
- 構図: **単体オブジェクトのみ、俯角30度の見下ろし(アイソメトリックゲーム視点)**。真横からの正面図はNG。**樹冠の上面が楕円状に見え、上の枝が下の枝より手前に重なって見える**こと。地面の影は落とさない(ゲーム側で処理)
- **立体感(必須)**: 平面的な図案・デカール調はNG。各葉叢は**厚みのある立体**として、上面=明・側面〜下面=暗の陰影を付け、**上の枝葉が下の枝葉に落とす影**を描く。幹は円柱の丸みが分かる陰影。手前の枝は明るく、奥の枝はやや暗く小さく(空気遠近)
- 背景: **単色(純白 #FFFFFF)** または透過。他のオブジェクト・地面・空を描かない
- 出力サイズ: 1024x1024以上(取り込みで縮小するため大きいほど良い)

## 樹種別プロンプト(コピペ用)

### 赤松(deco.tree.pine.*)
> 日本の赤松の一本木を斜め上30度から見下ろしたアイソメトリックゲーム用スプライト。曲がりくねった赤褐色の幹、水平に広がる枝、枝先に厚みのあるクッション状の松葉の傘。各傘は上面が明るく側面と下面が暗い立体的な陰影を持ち、上の傘が下の傘に柔らかい影を落とす。幹は円柱の丸みが分かる陰影。平面的な楕円の羅列や図案調にしない。戦国時代の屏風絵のような気品のある枝ぶり。ペインタリーな描き込み、落ち着いた暗めの暖色トーン、左上からの柔らかい光、純白の背景、単体、地面の影なし

### 杉(deco.tree.cedar.1)
> 日本の杉の一本木。まっすぐ高く伸びる幹、細く尖った円錐形の深緑の樹冠、層状に重なる枝葉。神社の参道の老杉の風格。ペインタリーなゲームアセット、落ち着いた暗めの暖色トーン、左上からの柔らかい光、やや見下ろしの視点、純白の背景、単体、影なし

### 広葉樹(deco.tree.broadleaf.1)
> 日本の欅(けやき)の一本木。太い幹から扇状に広がる枝、こんもりと丸い豊かな緑の樹冠、葉叢の明暗。ペインタリーなゲームアセット、落ち着いた暗めの暖色トーン、左上からの柔らかい光、やや見下ろしの視点、純白の背景、単体、影なし

### 竹叢(deco.bamboo.1)
> 日本の竹林の小さな一叢。細い青緑の稈が5〜8本、上部で笹の葉が羽のように茂る。ペインタリーなゲームアセット、落ち着いた暗めの暖色トーン、左上からの柔らかい光、やや見下ろしの視点、純白の背景、単体、影なし

### 藪(deco.bush.1)
> 日本の野の藪。低く茂る雑木と下草の塊、自然な乱れ。ペインタリーなゲームアセット、落ち着いた暗めの暖色トーン、左上からの柔らかい光、やや見下ろしの視点、純白の背景、単体、影なし
