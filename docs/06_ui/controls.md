# 操作一覧

## カメラ

- 矢印キー・中ボタンドラッグ・ミニマップクリック: スクロール
  (当初案のWASDは S=停止 / A=攻撃移動 と競合するため矢印キーへ変更。2026-07-04確定)
- マウスホイール: ズーム
- カメラ回転なし

## 選択

- 左クリック: 単体選択
- 左ドラッグ: 範囲選択
- Shift + 左クリック: 追加・除外
- ダブルクリック: 画面内同兵種を全選択 (実装済み 2026-07-05)
- Ctrl + 1〜9: グループ登録 (実装済み 2026-07-05)
- 1〜9: グループ呼び出し / 再押下でカメラジャンプ (実装済み 2026-07-05)

## 情報パネル

- 画面下部: 選択中ユニットの種別・HP・攻撃力・射程を表示 (実装済み 2026-07-05)
- 複数選択時: 種別ごとに集計して表示
- 建物クリック時: 種別・HP・兵糧/接続状態/門開閉を表示

## 命令

- 右クリック地面: 移動(複数選択時は陣形展開)
- 右クリック敵: 攻撃
- A + クリック: 攻撃移動
- S: 停止
- H: 位置固定(未実装)
- 自軍の門を左クリック: 開閉

## 工兵

- 梯子設置ボタン → 城壁際を指定
- 堀埋めボタン → 堀セルを指定

## 建築

- 建築ツールを選択してから地面を左クリックで配置
- **右クリック / Esc: 建築ツールをキャンセル → Select に戻る** (実装済み 2026-07-05)
- ドラッグ連続配置: 柵・壁・道・空堀・水堀は押下から離すまで通過したセルに連続配置 (実装済み 2026-07-05)
  - 検証 NG のセル (他建物・ユニット占有) は自動スキップ

## 画面レイアウト (3行ヘッダー, 実装済み 2026-07-05)

- 1行目: 速度制御 (⏸/1x/2x/4x) + 兵糧・金・武器・人口・招兵・年/季節
- 2行目: 城郭 (柵・壁・門・矢倉・本丸・天守) / インフラ (堀・道・橋) / 解体
- 3行目: 経済 (農地・蔵・市場・兵舎・武家屋敷・町区画) / 徴兵 / 工兵 / 市場取引 / セーブ・ロード / Debug

## 状態表示 (常時, 実装済み 2026-07-05)

- 左上 状態パネル:
  - 本丸保持残り時間 (シナリオに保持勝利条件がある場合)
  - 敵援軍接近カウントダウン (次の敵ウェーブがある場合)
- 上部中央バナー (緊急):
  - 敵兵站切断バナー (撤退タイマー動作中)
  - 兵糧危機バナー (蔵の残量が消費2サイクル分を下回った場合)
- 右クリックトースト: 戦術イベント (増援出現・梯子架設・兵糧少量・本丸接近)

## 区画

- 町区画・農地をブラシ指定
- 区画解除ツール

## 時間

- Space: 一時停止
- 速度ボタン: ⏸ / 1x / 2x / 4x
