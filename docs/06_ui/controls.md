# 操作一覧

## カメラ

- 画面端またはWASD: スクロール
- マウスホイール: ズーム
- カメラ回転なし

## 選択

- 左クリック: 単体選択
- 左ドラッグ: 範囲選択
- Shift + 左クリック: 追加・除外
- ダブルクリック: 画面内同兵種選択
- Ctrl + 1〜9: グループ登録
- 1〜9: グループ呼び出し

## 命令

- 右クリック地面: 移動
- 右クリック敵: 攻撃
- A + クリック: 攻撃移動
- S: 停止
- H: 位置固定

## 工兵

- 梯子設置ボタン → 城壁際を指定
- 堀埋めボタン → 堀セルを指定

## 建築

- 建築物選択
- 地面クリックで配置
- 右クリックまたはEscでキャンセル
- ドラッグ対応施設は連続配置

## 区画

- 町区画・農地をブラシ指定
- 区画解除ツール

## 時間

- Space: 一時停止
- 1倍、2倍、4倍のショートカットは技術設計時に確定
