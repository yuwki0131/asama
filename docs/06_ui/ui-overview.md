# UI概要

## 画面構成

```text
┌─────────────────────────────────────┐
│ 上部: 資源・兵糧・人口・時間・警告 │
├───────┬─────────────────────┬───────┤
│ 左側  │                     │ 右側  │
│ 建築  │     ゲーム画面      │ 詳細  │
│ 区画  │                     │ 取引  │
├───────┴─────────────────────┴───────┤
│ 下部: 選択ユニット・命令・ミニマップ│
└─────────────────────────────────────┘
```

左右パネルは折り畳み可能です。

## 上部バー

### 資源

- 銭
- 木材
- 石材
- 武器

### 兵糧

防衛側:

- 利用可能兵糧
- 全蔵総量
- 未接続兵糧
- 推定維持時間

攻撃側:

- 遠征兵糧
- 補給荷車数
- 推定維持時間

### 人口・時間

- 人口 / 上限
- 支持率
- 徴兵可能人数 / 最大
- 年・季節
- 一時停止、1倍、2倍、4倍

## 左側

- 城郭建築
- 内政建築
- 区画指定

## 右側

選択中の建物、区画、本丸、市場、兵舎等の詳細を表示します。

## 下部

- 単体・複数ユニット情報
- 命令ボタン
- 兵種構成
- ミニマップ
