## デバッグステータスパネル

開発中の状態確認用に、読み取り専用のデバッグステータスパネルを持ちます。

表示フラグ:

- `VITE_DEBUG_STATUS_PANEL=true`: 表示する
- `VITE_DEBUG_STATUS_PANEL=false`: 表示しない
- 未指定時は開発サーバー実行中のみ表示する

MVP実装ではゲーム画面右上に重ねて表示します。プレイヤー入力を奪わないよう、パネルはクリック対象にしません。

表示項目:

- デバッグ表示フラグ状態
- シミュレーション状態
- 現在tick
- マップサイズ
- 現在の建築ツール
- ユニット数
- 選択ユニット数
- 先頭選択ユニットID
- 先頭選択ユニット座標
- 先頭選択ユニットの経路長
- 移動または建築の無効対象座標
- 建築数
- 建築タイプ別カウント
- 直近のシミュレーションエラー

このパネルは正式HUDではなく、開発・検証用です。プレイヤー向けUIとして必要になった情報は、個別にHUDや詳細パネルへ移します。
