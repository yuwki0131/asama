# 戦闘システム

## 基本

- HP制
- HPが0以下で死亡
- 士気なし
- 疲労なし
- 自動退却なし
- 負傷による性能低下なし

## ダメージ

概念式:

```text
最終ダメージ = 基礎攻撃力 × 兵種相性 × 高低差補正 × 対象種別補正
```

小さな乱数幅を入れる場合も±5〜10%程度に留めます。

## 兵種相性

装備個別管理ではなく、兵種カテゴリへの倍率で表現します。

例:

- 槍足軽は騎兵に強い
- 騎兵は弓兵・鉄砲兵に接近しやすい
- 鉄砲兵は高HP・騎兵への単発威力が高い
- 工兵は直接戦闘が弱い

## 高低差

高所側を有利にします。

- 近接: 攻撃力へ軽い補正
- 弓・鉄砲: 攻撃力または射程へ補正
- 低所側: 軽い不利

具体倍率はデータ定義で調整します。

## 近接戦

- 上下左右の4方向隣接で攻撃可能
- 斜め隣接では原則攻撃不可
- 敵ユニットのセルへ進入不可
- 1対象へ同時に直接攻撃できる人数は最大4人
- 同一方向に複数味方が重なっていても代表1人だけ攻撃

## 遠距離戦

攻撃条件:

- 射程内
- 射線が通る
- 攻撃間隔完了
- 射撃許可状態

MVPでは命中率中心のシステムにせず、条件成立時に着弾ダメージを与えます。

## 射線遮蔽

遮蔽対象:

- 地形
- 石垣・土塁
- 建物

味方ユニットは射線を遮らないものとします。

## 建物攻撃

- 門は一般近接兵で攻撃可能
- 工兵の門攻撃補正なし
- 石垣・土塁は通常兵で実質破壊不能
- 遠距離兵の門攻撃は原則無効

## 建物内戦闘

対象:

- 門
- 櫓
- 天守
- 本丸内施設

マップは単層です。建物内部へ入ると屋根・上部を透過し、ゲーム用に圧縮された単一内部空間で戦います。
