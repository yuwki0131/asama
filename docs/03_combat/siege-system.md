# 攻城システム（MVP）

## MVP対象

- 梯子
- 門破壊
- 堀埋め

以下はMVP外です。

- 火災
- 火矢
- 直接放火
- 大筒
- 輸入大砲
- 破城槌
- 投石器
- 仮設橋

## 梯子

### 設置

- 工兵を選択
- 城壁・石垣際の設置地点を指定
- 工兵が地点まで移動
- 短い設置時間後に梯子生成
- 資源消費なし

### 利用

- 設置後は一般歩兵も通行可能
- 単層マップ上の特殊接続として扱う
- 外側セルと内側セルを接続

### 破壊

- 守備側の近接歩兵が攻撃可能
- HPゼロで消滅
- 消滅後は接続リンク削除

## 門破壊

- 近接兵が門に隣接して通常攻撃
- 槍足軽、刀足軽、侍、騎兵、工兵が実行可能
- 工兵専用補正なし
- 遠距離兵は原則無効
- HPゼロで破壊
- 破壊後は通行可能

## 堀埋め

- 工兵を選択
- 対象の空堀・水堀セルを指定
- 工兵が隣接地点まで移動
- 所定時間の作業を行う
- 資源消費なし
- 完了後、埋立地へ変化

### 中断

- 工兵死亡
- 別命令
- 対象消失

進捗保持の有無はパラメータ化し、初期値は保持を推奨します。

```text
fillDurationTicks
preserveProgressOnInterrupt
resultTerrainType
```

## 石垣・土塁

- 通常歩兵攻撃では破壊不能に近い
- MVPでは実質無敵として扱ってよい
- 攻撃側は門、梯子、堀埋めで攻略する
