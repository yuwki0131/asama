# 経路探索・占有・衝突

## 基盤

- 論理地形: 正方形グリッド
- 表示位置: 連続座標
- 経路探索: 地形コスト付きA*
- 多層ナビゲーションなし

## 占有サイズ

歩兵、騎兵、工兵、補給荷車をすべて同一サイズとして扱います。

見た目のサイズ差は経路探索に影響しません。

## 味方ユニット

- 同一セルへの重複可能
- 移動中は重複を許容
- 経路探索上の障害物にしない
- 停止後、短時間重複が続けば近傍の空きセルへ分散
- 空きセルがなければ重複継続

分散先は同一接続領域内をBFSで探索し、壁や閉門を飛び越えません。

## 敵ユニット

- 敵が存在するセルは通行不可
- 敵を経路探索上の恒久障害には含めず、次セル進入直前に判定
- 通常移動では停止
- 攻撃移動では攻撃し、排除後に再探索

## 経路計算タイミング

1. 移動命令時にA*実行
2. 経路を保持して移動
3. 次セルへ進入直前に通行可否確認
4. 固定障害なら再探索
5. 敵ユニットなら停止

## 動的障害

即時再探索対象:

- 閉じた門
- 新設された壁・建物
- 破壊された橋
- 通行不能な瓦礫
- 将来の火災セル
- 梯子消失

味方による一時的混雑では再探索しません。

## 到達不能

- 数回再探索
- 経路がなければ目的地に最も近い到達可能セルへ移動
- 停止して到達不能を通知

## 地形コスト

暫定値:

| 地形 | コスト |
|---|---:|
| 道路 | 0.75 |
| 平地・草地 | 1.00 |
| 水田 | 1.25 |
| 森林 | 1.50 |
| 緩斜面 | 1.25 |
| 急斜面 | 2.00 |
| 湿地 | 2.50 |
| 空堀 | 2.00 |
| 瓦礫 | 1.75 |
| 河川・水堀・崖 | 通行不可 |

## 高低差

- 高さ差0〜1: 通行可能
- 高さ差2: 高コスト
- 高さ差3以上: 原則通行不可
- 特殊接続は門、橋、梯子等で表現

## 門・通路幅

- 小門: 1セル
- 通常門: 2セル
- 大型門: 3セル
- 櫓・天守内部: 基本1セル幅
- 梯子: 1セル幅

重複可能なため、通行予約キューは実装しません。
