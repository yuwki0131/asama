# ユニット操作

## 基本方針

兵士は一人単位で選択・操作します。固定部隊オブジェクトや厳密な陣形はありません。

## 選択

- 左クリック: 単体選択
- ドラッグ: 範囲選択
- Shift + クリック: 追加・除外
- ダブルクリック: 画面内の同兵種選択
- Ctrl + 1〜9: 操作グループ登録
- 1〜9: 操作グループ呼び出し

## 右クリック命令

| 対象 | 命令 |
|---|---|
| 地面 | 移動 |
| 敵ユニット | 攻撃 |
| 敵門・建物 | 建物攻撃 |
| 梯子 | 利用 |
| 蔵 | 防衛側荷車の格納 |
| 堀 | 工兵による堀埋め |

## 複数移動

クリック地点を中心に、選択数に応じた目的地セル群を作り、各ユニットへ別セルを割り当てます。

- 移動中の陣形維持なし
- 元の相対位置維持なし
- 到着地点だけ分散
- 通行可能セルが不足すれば範囲拡張

## 通常移動

- 目的地を優先
- 敵グリッドへは進入不可
- 敵と隣接した場合は自動応戦可能
- 敵による停止時は自動迂回せず、プレイヤー再指示を基本とする

## 攻撃移動

- 移動中に敵を発見すると攻撃
- 敵排除後、元目的地へ再探索

## 位置固定

- 現在地点から大きく移動しない
- 射程内の敵には攻撃
- 追跡しない

## 遠距離兵

- 自動射撃
- 射撃停止

一斉射撃など高度な射撃制御はMVP外です。

## 住民

住民は直接選択・命令できません。町区画、農地、道路、施設配置を通じて間接制御します。
