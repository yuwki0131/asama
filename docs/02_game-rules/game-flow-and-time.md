# ゲーム進行と時間

## 全体方針

1プレイは1倍速換算で約90分です。築城、城下町造営、攻城戦を約3サイクル繰り返します。

## 暦時間

| 単位 | 現実時間（1倍速） |
|---|---:|
| 1か月 | 75秒 |
| 1季節 | 3分45秒 |
| 1年 | 15分 |
| 1プレイ | 約6年 |

## 戦術時間

次の処理は通常RTS相当の戦術時間で進行します。

- ユニット移動
- 攻撃・射撃
- 門破壊
- 梯子設置
- 堀埋め
- 補給荷車の移動

## 内政時間

次の処理は暦サイクルで更新します。

- 徴税
- 支持率
- 人口増減
- 徴兵可能人数回復
- 町区画成長
- 田植え判定
- 秋の収穫

## 推奨90分構成

### 第1サイクル: 0〜30分

- 基礎築城
- 初期町区画・農地整備
- 小規模攻城戦

### 第2サイクル: 30〜68分

- 外郭拡張
- 人口・税収・兵力拡大
- 中規模攻城戦

### 第3サイクル: 68〜90分

- 最終防御準備
- 本丸周辺強化
- 最終攻城戦

実際の時刻・攻撃回数はシナリオデータで設定します。

## 速度操作

- 一時停止
- 1倍
- 2倍
- 4倍

シナリオ上の経過時間はゲーム内シミュレーション時間で管理します。

## 周期処理

| 処理 | 周期 |
|---|---|
| 戦闘・移動 | 毎tick |
| 本丸―蔵接続 | 約8〜12秒のseed付き揺らぎ |
| 兵糧消費 | 接続判定周期で一括処理 |
| 人口・税収・徴兵枠 | 毎月 |
| 季節変更 | 3分45秒ごと |
| 収穫 | 秋終了時 |

接続判定の正確な次回時刻はUIに表示しません。
