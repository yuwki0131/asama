# 建築・解体

## 即時建築

建築は経済・内政時間に属する抽象処理とし、配置時点で即時完成します。

1. 配置可否判定
2. 資源を全量消費
3. 完成状態で出現
4. 最大HPで機能開始

実装しないもの:

- 建設時間
- 建設作業員
- 建築進捗
- 建築途中モデル
- キャンセル

## 敵付近での建築禁止

敵ユニットから一定グリッド以内では建築できません。距離は共通パラメータとして調整可能にします。

この制限により、接敵地点で壁や門を即時生成する行為を防ぎます。

## 解体

- 即時
- 資源返却なし
- 敵付近では不可
- 内部に自軍ユニットがいる建物は不可
- 蔵解体時は食料全損
- 本丸は解体不可

## 平時建築と攻城作業

即時建築:

- 柵、塀、門
- 土塁、石垣
- 空堀、水堀
- 土橋、木橋
- 櫓、天守
- 内政建物

戦術行動:

- 梯子設置
- 堀埋め
- 門破壊

## 本丸設定

### 通常マップ

- ゲーム開始時にプレイヤーが設定
- 連続セル集合
- 開始後変更不可
- 配置場所は自由
- 面積上限はシナリオまたはルール設定

### シナリオマップ

- マップ側で事前設定可能
- プレイヤー変更不可

## 天守

- 本丸内に配置
- 必要資源があれば即時完成
- 高コストによって建築までの経済時間を表現
- 勝敗条件ではない
