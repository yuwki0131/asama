# 勝利・敗北条件

## 防衛側の敗北

### 本丸陥落

次の条件を同一シミュレーション断面で満たした場合、本丸陥落です。

- 本丸内の防衛側占領可能ユニットが0体
- 本丸内の攻撃側占領可能ユニットが1体以上

占領可能ユニットに含むもの:

- 近接兵
- 遠距離兵
- 工兵
- 騎兵

含まないもの:

- 住民
- 補給荷車
- 無人設備
- 投射物

### 兵糧切れ

本丸から利用可能な蔵の食料を、兵糧消費周期で必要量だけ支払えなかった場合、開城して敗北します。

一瞬ゼロになっただけでは即時敗北せず、次回の兵糧判定断面で確定します。

## 防衛側の勝利

シナリオごとに以下を組み合わせます。

- 規定時間まで本丸を保持
- 攻撃側戦闘ユニット全滅
- 攻撃側兵糧ゼロ

## 攻撃側の勝利

- 本丸占領
- 防衛側兵糧ゼロ

## 攻撃側の敗北

- 攻撃側戦闘可能ユニット全滅
- 攻撃側補給荷車の食料総量ゼロ
- シナリオ固有の制限時間超過

## 天守・城主

- 天守破壊は直接の勝利条件ではない
- 城主死亡もMVPでは独立勝敗条件にしない
- 城主は演出上、本丸内に存在してよい
