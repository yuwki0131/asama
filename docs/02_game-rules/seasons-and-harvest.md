# 四季・田植え・収穫

## 四季

MVPでは四季の直接効果は秋の収穫だけです。

- 春: 田植え判定
- 夏: 特別効果なし
- 秋: 収穫期間
- 秋終了時: 一括収穫
- 冬: 特別効果なし

天候、雨、雪、風、季節移動補正はMVP外です。

## 田植え判定

春の所定時点で存在する農地を、その年の収穫対象として登録します。

- 田植え判定前に設置した農地: 当年収穫対象
- 判定後に設置した農地: 翌年から対象
- 収穫前に自軍が撤去した農地: 対象外
- 敵は農地を破壊できない

## 通常時の収穫格納

秋終了時、収穫量を本丸から近い蔵の順に格納します。

1. 本丸から各蔵への距離順位を取得
2. 現在到達可能な蔵だけ抽出
3. 近い蔵から容量上限まで格納
4. 全蔵が満杯なら余剰は失われる

蔵順位は次のイベントで再計算します。

- 蔵新築
- 蔵解体
- 蔵破壊
- 本丸設定
- マップロード

収穫処理時には現在の到達可否を再確認します。

## 攻城中の収穫

攻城状態では、防戦中の城内蔵へ自動搬入しません。

1. マップ外部搬入口から到達可能な蔵を抽出
2. 対象蔵へ均等配分
3. 容量不足分を残り蔵へ再配分
4. すべて満杯なら余剰を失う

本丸との接続はこの格納判定には使用しません。

## 外部支援

シナリオイベントとして兵糧を追加できます。

- 防衛側: 補給荷車として出現し、蔵到着後に兵糧化
- 攻撃側: 補給荷車として出現時点から兵糧化

## 市場

収穫以外にも市場で兵糧を購入・売却できます。これによりプレイヤーは兵糧増減をある程度制御できます。
